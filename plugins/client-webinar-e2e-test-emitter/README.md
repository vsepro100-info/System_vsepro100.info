# Client Webinar E2E Test Emitter

**Статус:** verification-only, removable.

## Назначение
Плагин предназначен только для минимальной E2E‑проверки клиентского вебинарного пайплайна. Он регистрирует временный `admin-ajax`-хук и эмитит **raw-сигнал** `client_webinar_entered` без UI, таймеров, персистентности или побочных эффектов.

## Что делает
- Регистрирует `admin-ajax` действие `__e2e_webinar_entered`.
- Эмитит raw-сигнал `client_webinar_entered` через `do_action`.
- **Не** эмитит канонические события напрямую.

## Формат входных данных
Отправьте `POST` на `admin-ajax.php?action=__e2e_webinar_entered` со следующими параметрами:
- `lead_id` (int, опционально)
- `webinar_id` (string, опционально)
- `timestamp` (int, опционально; если не передан, используется текущее время)

В контекст эмиссии также добавляется `source: client-webinar-tracker-v2` для соответствия архитектурным ограничениям.

## Как удалить после проверки
1. Деактивируйте плагин в WordPress.
2. Удалите каталог `plugins/client-webinar-e2e-test-emitter`.

После удаления E2E‑триггер будет недоступен.
