# ADR-001: LIVE vs AUTO вебинары

## Решение
LIVE и AUTO — разные независимые модули.

## Причины
- Разные риски.
- Разные режимы доставки.
- Стабильность продакшена важнее скорости изменений.

## Общий слой
- Только ref/CRM/аналитика через контракты.

## Последствия и запреты
- Не объединять LIVE и AUTO.
- Не создавать вторую CRM.
- Не трогать /signup/.
