# Книга проекта (Knowledge Base)

## Как пользоваться книгой
1. Начните с этого оглавления и перейдите по ссылкам ниже.
2. Любой новый факт/процесс сначала фиксируется здесь, затем выполняется.
3. Если в книге нет записи — считается, что этого в системе нет.
4. Для изменений используйте соответствующий раздел (решения, деплой, проблемы).
