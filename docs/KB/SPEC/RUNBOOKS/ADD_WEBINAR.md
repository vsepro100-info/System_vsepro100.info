# Runbook: добавить вебинар

## Цель
Добавить/обновить вебинар в расписании через каноническую админ‑страницу, не нарушая архитектурные запреты.

## Предусловия
- Есть права доступа к `/admin_webinar/`.
- Согласованы правила CTA и доступа (см. [CANON/ARCHITECTURE](../../CANON/ARCHITECTURE.md)).

## Шаги
1. **Откройте админ‑страницу**
   - URL: `/admin_webinar/`.
   - Проверьте, что вы в режиме Speaker UI.
2. **Заполните/обновите ключевые поля**
   - Заголовок, дата/время.
   - Постер.
   - iframe плеера.
   - iframe чата.
   - CTA (гостевой CTA по умолчанию — `/signup/`).
3. **Проверьте публичный список**
   - Страница `/webinars/` показывает актуальное расписание.
4. **Проверьте комнату вебинара**
   - Страница `/account/webinar_room/` корректно отображает состояние.

## Контрольные запреты
- Не трогать /signup/.
- Не создавать вторую CRM.
- Не объединять LIVE и AUTO.
