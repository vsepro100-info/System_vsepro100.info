# Webinars — Roadmap v2 (кратко)

## Назначение
Коротко зафиксировать вектор развития после MVP v1+. Без детализации реализации и без привязки к срокам.

## Roadmap v2 (3–5 пунктов)
1) **Приоритет: AUTO‑контур вебинаров** — запуск и доставка AutoWebinar как отдельный runtime‑контур.
2) Регистрация на вебинар как каноническое событие (`client_webinar_registered`).
3) Полные события участия: `client_webinar_left`, `client_webinar_form_submitted`.
4) Расширение post‑webinar сценариев (каналы и маршруты, помимо Telegram).
5) Доступ к записям для партнёров (после LIVE‑контента).
