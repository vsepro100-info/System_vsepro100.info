# Архив KB

Архив содержит **сырые исторические материалы без структуры** и документы, утраченные как канон.
Используйте их как справку, но не как источник текущей истины.

См. также:
- `MIGRATED_NOTES.md` — объяснение, почему документы перенесены или заменены.
- `RAW/` — несистематизированные исторические материалы (черновики, бинарные артефакты и т.п.).
